# Real-Time Object Detection Using Hardware-Accelerated CNN on Xilinx Zynq FPGA with Arm Processor
## Overview
  This project presents the design and implementation of a real-time Convolutional Neural Network (CNN) inference accelerator on a Xilinx Zynq System-on-Chip (SoC) platform. The architecture leverages hardwareâ€“software co-design by offloading computationally intensive CNN layers to the FPGA fabric while utilizing the ARM Cortex-A9 processor for system control, preprocessing, and post-processing tasks.

  A custom hardware-accelerated CNN IP core was developed using High-Level Synthesis (HLS) in Xilinx Vitis HLS, integrated into Xilinx Vivado, and deployed on the PYNQ-Z2 development board. The implementation demonstrates significant performance improvements over a CPU-only baseline, enabling efficient real-time image processing.
## Key Features
* FPGA-accelerated CNN inference
* Hardware/Software co-design using Vitis and Vivado
* Real-time image classification/object detection
* Quantitative comparison with CPU-only implementation

## Hardware Platform
* Xilinx Zynq SoC (PYNQ-Z2)
* ARM Cortex-A9
* FPGA Fabric with DSP acceleration
## Software Stack
* Vivado Design Suite
* PYNQ Framework
* Python + OpenCV
